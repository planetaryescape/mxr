## Go-to navigation (Gmail-style)

| Key | Action |
|-----|--------|
| `gi` | Go to Inbox |
| `gs` | Go to Starred |
| `gt` | Go to Sent |
| `gd` | Go to Drafts |
| `ga` | Go to All Mail |
| `gl` | Go to Label (picker) |

## Actions

| Key | Action |
|-----|--------|
| `Enter` / `o` | Open message |
| `/` | Search |
| `n` / `N` | Next / previous search result |
| `Ctrl-p` | Command palette |
| `Esc` | Back / close |
| `q` | Quit |
