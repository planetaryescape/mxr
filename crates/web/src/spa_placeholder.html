The bridge is running but the embedded SPA files are missing. Build the web app first: 

cd apps/web
npm install
npm run build
cargo build --features web-ui --release
mxr web

For local development, prefer running the Vite dev server directly: 

cd apps/web
npm run dev          # http://localhost:5173
